Defanged Indicators Test Page

This page contains defanged (obfuscated) indicators that the extension should detect and refang.

Defanged IP Addresses

  • Using brackets: 192[.]168[.]1[.]1
  • Using hxxp prefix: hxxp://10[.]0[.]0[.]1/payload
  • Mixed defanging: 8.8[.]8.8

Defanged Domains

  • Full brackets: evil[.]example[.]com
  • Partial brackets: malware.evil[.]net
  • With protocol: hxxps://phishing[.]site[.]org/login

Defanged URLs

  • hxxp://malware[.]com/dropper.exe
  • hxxps://c2-server[.]net:8443/beacon
  • hxxp[:]//encoded[.]url[.]com/path

Defanged Email Addresses

  • attacker[@]evil[.]com
  • phisher[at]malicious[.]org

The extension should correctly identify and refang all these indicators.
